About
-----

With this app you can control multiple mpv players in one system.


Scheme
------
```
+----------------------------------------+
|  +------+                              |
|  | mpv1 |----+        Computer 1       |
|  +------+    |                         |
|              |                         |
|  +------+    |    +-----------------+  |  +---------+
|  | mpv2 |----+----| MPV HTTP Router |--|--| MPV Web |
|  +------+    |    +-----------------+  |  +---------+
|              |                         |
|  +------+    |                         |
|  | mpv3 |----+                         |
|  +------+                              |
+----------------------------------------+
```
